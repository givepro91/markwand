Markwand 처음 실행 안내

무료 배포판은 Apple 공증을 사용하지 않기 때문에 첫 실행 때 macOS가 한 번 확인합니다. 최신 macOS에서는 우클릭 → 열기로도 막힐 수 있어, 이 ZIP에는 첫 실행 헬퍼를 함께 넣었습니다.

  1. Markwand.app을 Applications 폴더로 드래그합니다.
  2. Applications 폴더에서 Markwand를 Control-클릭 또는 우클릭 → 열기를 먼저 시도합니다.
  3. 사진처럼 휴지통으로 이동 / 완료만 보이면 창을 닫습니다.
  4. 이 폴더의 Open Markwand.command를 더블클릭합니다.
  5. Terminal 창이 잠깐 열리며 Markwand를 실행합니다.

이 과정은 처음 한 번만 필요합니다. 다음부터는 Markwand를 더블클릭해서 실행할 수 있습니다. 완전히 매끄러운 설치는 Apple Developer ID 서명과 공증이 필요합니다.
