## 🌐 다국어 지원 (i18n)

- **자동 감지**: 시스템 언어가 `ko-*` 이면 한국어, 그 외 **영어**
- **수동 전환**: 설정(⚙) → **Language** / **언어** → 한국어 / English 선택
- 선택 즉시 반영 (앱 재시작 불필요)
- 선택 값은 prefs 저장 → 다음 실행 시 유지

### 번역된 화면
- SSH 원격 서버 추가 모달 (폼·폴더 picker·에러 메시지)
- SSH 호스트 키 신뢰 모달 (TOFU)
- 설정 팝오버 (전체)
- 워크스페이스 picker (드롭다운·빈 상태)
- 빈 상태 / 로딩 오버레이
- 파일 트리 로딩 안내

### 향후 (점진 확장)
- ProjectView, FileTree, MarkdownViewer 등 내부 UI 라벨 (현재 한국어 하드코딩 유지)
