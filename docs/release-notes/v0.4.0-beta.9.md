# v0.4.0-beta.9 — production launch guard

> 2026-05-01 · macOS prerelease (free ZIP, ad-hoc signed)

beta.9 fixes a production launch bug where a packaged app could still honor development environment variables.

## Fixed

- Packaged Markwand now ignores `ELECTRON_RENDERER_URL`.
- Packaged Markwand now ignores `MD_VIEWER_DEBUG`.
- DevTools no longer auto-opens in packaged builds because of shell environment leakage.
- The packaged app always loads its bundled renderer `index.html`.

## Verification

- Unit regression test for packaged runtime mode.
- `pnpm typecheck`
- `pnpm test`
- `pnpm dist:mac:free`
- Local installed app smoke with `ELECTRON_RENDERER_URL=http://127.0.0.1:9` and `MD_VIEWER_DEBUG=1` forced in the launch environment.
