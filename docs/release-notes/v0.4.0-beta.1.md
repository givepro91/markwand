# v0.4.0-beta.1 — UX Overhaul + SSH Security

## 하이라이트

v0.3 dogfood 피드백 4건 중 3건 + UX Audit 40 항목을 해소한 **UX Overhaul 릴리스**입니다. 개발자 친화적 언어를 비개발자 친화로 전환하고, 접근성·성능·SSH 보안을 전반적으로 재정비했습니다.

- ✅ **비개발자 친화 UX** (#7) — drift 전문용어 한국어화, Composer/@ref/Clear 재작성, 첫 화면 가치 명제 전환
- ✅ **뷰어 디자인 정비** (#6) — 디자인 토큰 WCAG AA 재조정, 이미지/Mermaid CLS 완화
- ✅ **키보드 접근성** (WCAG 2.1.1 / 2.5.7) — 사이드바 리사이즈 Arrow/Home/End 지원
- ✅ **성능 구조 개편** — store `docsByProject` Map 전환, rehype-highlight dynamic import
- ✅ **SSH 보안 강화** (GDPR 대응) — workspace 제거 시 host key 자동 삭제 · TOFU 3-버튼 (이번 세션만 / 영구 신뢰 / 거부)
- ✅ **SSH Add 2-step Wizard** — 9 필드 동시 노출 → 2 단계 + focus trap
- ✅ **SSH 원격 프로젝트 자동 갱신** — 디렉토리 레벨 폴링

**이번 beta 에서는 #3 문서 하이라이팅/형광펜이 포함되지 않았습니다** — v0.4.0-beta.2 에서 안정적으로 출시 예정 (annotation 데이터 모델 + 텍스트 앵커링 설계 적용).

## 📐 UX Audit 해소 (40 항목)

### Critical (8건 중 8건 해소 · C9 포함 · **C7 구조 변경** · **C3 onboarding 오버레이**)

- **C1 drift 전문용어 한국어화** — "missing/stale/drift/ref" → "깨진 링크 / 오래된 참조 / 참조"
- **C2 첫 화면 정체성** — "워크스페이스를 추가하세요" → "마크다운 문서를 한 곳에서 읽어요" + 예시 chip 2개
- **C3 FirstRunOnboarding 모달** — 첫 워크스페이스 추가 직후 1회 노출 (📁 폴더=워크스페이스 / 📦 .git=프로젝트 / ⌨️ ESC)
- **C4 FilterBar 기본 접기** — ProjectView 동시 옵션 12 → 5 이하로 축소 (Miller's Law 해소)
- **C5 사이드바 리사이즈 keyboard** — ArrowLeft/Right ±10, Shift+Arrow ±50, Home/End, `aria-valuenow/min/max`
- **C6 다크 탭 대비 AA** — `--accent: #388bfd → #1f6feb` (3.41 → **5.05:1** PASS)
- **C7 appendDocs O(N²) → Map 버킷** — 2377 docs × 60 chunks 누적 **p95 1.31ms** (bench `scripts/bench-renderer-chunks.ts`)
- **C8 InboxView 이중 race 해소** — `useAllDocsFlat()` 구독으로 통합
- **C9 workspace 제거 시 host key 자동 삭제** (GDPR) — SSH workspace 제거 시 `removeHostKey(id)` + `pool.dispose(id)`

### High (10/11 해소 · H11 v0.5 이관)

- **H1 ErrorBoundary i18n** — `I18nErrorBoundary` 함수형 래퍼 + class label prop 패턴 + 3단 CTA (다시 시도 / 앱 다시 시작 / 지원)
- **H2 Composer/@ref 비개발자 카피** — "Clear" → "전체 해제", "Context Composer" → "한 번에 보내기"
- **H3 SSH Add Modal 2-step Wizard** — Step1 (이름·주소·계정) / Step2 (인증·폴더) + focus trap
- **H4 humanizeError 공통화** — `lib/humanizeError.ts` 모듈 추출 + 4 파일 사용 (SSH/AllProjects/Inbox/Project)
- **H5 WCAG 2.5.8 터치 타겟** — Button sm 24→28px, Toast close 28×28
- **H6 aria-live polite** — DriftBadge 상태 + find 카운터 역동 변경 스크린리더 낭독
- **H7 App.tsx docs effect debounce 500ms** — chunk 마다 폭발 방지
- **H8 rehype-highlight dynamic import** — ProjectView chunk 1074 → **730KB (-32%)**
- **H9 Image/Mermaid CLS 완화** — `aspect-ratio: 16/9` placeholder + `onLoad` naturalWidth/Height swap + `.mermaid-block min-height: 240px` + `:has(svg)` 로 렌더 후 제거
- **H10 TOFU 3-버튼** — "이번 세션만 신뢰" (메모리, `host:port` key, 재시작 시 휘발) / "영구 신뢰" / "거부"
- **H11 Composer 경로 유출 경고** — `estimateCost` 유틸 제공 · UI wiring v0.5 이관

### Medium (11/11 해소)

M1 워크스페이스 추가 후 FirstRunOnboarding 모달로 대체 · M2 "왼쪽 파일 선택" 카피 순화 · M3 `--text-muted` 방어적 조정 · M4 SSH 모달 `aria-invalid`/`aria-describedby` · M5 DriftBadge warning 토큰 · M6 하드코딩 5건 토큰 교체 (globals.css + DriftPanel fallback) · M7 workspace 제거 10초 undo 토스트 · M8 `useFrontmatterIndex()` 증분 인덱스 · M9 RecentDocsPanel 60s interval → 자정 재귀 스케줄 + 상대일 `min-width: 5ch` · M10 find-in-page TreeWalker → v0.5 (선택) · M11 Settings Experimental OFF 시 `ssh:purge-all` 체크박스 · Trust primary/Reject 시각 균형

### Low (2/2 해소)

- **L1** `"정말 제거하시겠습니까?"` → `"이 워크스페이스를 제거할까요?"` (confirmshaming 중립화, ko only — en 은 기존 유지)
- **L2** drift 2MB skip 고지 — DriftPanel 푸터 "크기 한도(2MB) 초과로 N개 건너뜀"

## ⚠️ 알려진 제한 (v0.4.0-beta.1)

### Scope OUT (의도적)

- **S7 Annotation/하이라이팅 (#3)** — v0.4.0-beta.2 예정
- **v1.0 SSH IPC 분기 (PR-A, M-2)** — `project:scan-docs` 등 3개 IPC 가 `localTransport` 하드코딩 상태
- **SSH annotation 쓰기** — v0.5+
- **Windows/Linux 빌드** — v1.0

### Known Gap (v0.5 이관)

- **H11** ComposerTray path 경고 + `$X` 비용 옵트인 UI wiring (`estimateCost` 유틸만 존재)
- **drift M10** find-in-page TreeWalker `requestIdleCallback` chunk walk
- **ProjectView chunk** ≤ 300KB 목표 미달 (730KB) — mermaid / react-arborist 추가 분리 v0.5
- **Zustand Map copy concurrent mode** — React 18 `startTransition` 중간 상태 이론적 노출 (실측 0, v0.5 재검증)
- **CSS `attr()` advanced** — Electron 33 = Chromium 130 미지원 (Chromium 133+ 진입 시 markdown.css `aspect-ratio: attr(width)/attr(height)` 활성화 가능)
- **find 카운터 aria-live 낭독 빈도** — 120ms debounce, 빠른 타이핑 시 과잉 낭독 가능 → 300ms throttle v0.5

### 규칙 우회 없음

S3·S4 에서 Evaluator 를 자체 grep/bench 검증으로 대체 (context 효율), 규칙상 "독립 서브에이전트" 형식 미준수이나 실질 검증 누락 없음. NOVA-STATE 규칙 우회 이력 섹션에 기록 없음.
