# v0.4.0-beta.7 — macOS first-run helper

> 2026-05-01 · macOS prerelease (free ZIP, ad-hoc signed)

beta.7 is a packaging-only follow-up to beta.6. The app code is unchanged except for the version number.

## Why

On recent macOS versions, a free/ad-hoc signed app downloaded from the browser can still show **Move to Trash / Done** even when launched with right-click → Open. beta.6's install guide was too optimistic for that case.

## Changed

- Added `Open Markwand.command` to the ZIP.
- The helper removes the quarantine flag from `/Applications/Markwand.app` and opens the app.
- Updated `First Run Guide.html` and macOS install docs to explain this path.

## Verification

- `pnpm typecheck`
- `pnpm test`
- `pnpm build`
- `pnpm dist:mac:free`
- ZIP contains `Markwand.app`, `First Run Guide.html`, and `Open Markwand.command`
- `codesign --verify --deep --strict`
- Local install smoke after quarantine removal
