# v0.4.0-beta.10 — faster quit guard

beta.10 fixes a quit-path regression found during local install QA.

## Fixed

- Packaged apps now hide all windows immediately on quit, so Cmd+Q and Finder quit no longer feel stuck while cleanup runs.
- The delayed startup watcher timer is cancelled during quit, preventing a race where the app could start file watching while already shutting down.
- Quit cleanup watchdog is capped at 500ms. If watcher or SSH cleanup stalls, the process exits instead of making the user wait.
- Startup file watching is now opt-in (`MARKWAND_ENABLE_STARTUP_WATCH=1`) because large workspace crawls could delay quit and spike CPU.

## QA

- `pnpm typecheck`
- `pnpm test` — 39 files, 379 tests
- Targeted quit cleanup regression tests
- Local `/Applications/Markwand.app` packaged smoke with forced dev env vars
- Local quit timing and no-orphan-process check
