# v0.4.0-beta.6 — Project Wiki + 무료 ZIP 배포 + 종료 안정화

> 2026-05-01 · macOS prerelease (free ZIP, ad-hoc signed)

---

## 하이라이트

beta.6은 Markwand를 “많아지는 AI 산출물 문서를 이해 가능한 프로젝트 위키로 바꾸는 앱” 쪽으로 크게 밀어낸 릴리스입니다. 동시에 앱 아이콘, 전체 UI 톤, 무료 macOS 설치 경로, 종료 안정성까지 릴리스 전 QA에서 확인된 문제를 정리했습니다.

- **Project Wiki MVP** — 프로젝트의 README, 설계 문서, 리스크, 결정 로그, 지식 클러스터를 자동 요약
- **Start Here / Risk Board / Knowledge Map** — 처음 읽을 문서, 위험한 문서, 문서 묶음을 한 화면에서 안내
- **AI Task Suggestions** — 끊긴 참조 수정, 오래된 문서 갱신, 메타데이터 보강 등 바로 복사 가능한 작업 프롬프트 제공
- **문서에서 Wiki로 돌아가기** — 문서를 읽다가 프로젝트 위키로 복귀하는 내부 내비게이션 추가
- **동일 파일명 표시 버그 수정** — `CLAUDE.md`처럼 basename이 같은 문서는 상대 경로로 구분
- **새 앱 아이콘 + 리프레시된 UI 컨셉** — warm paper / ink / mint 계열의 더 부드러운 제품 톤 적용
- **무료 macOS ZIP 배포** — Apple Developer Program 없이 `Markwand-<version>-<arch>-free.zip` 생성
- **종료 지연 방지** — 종료 cleanup이 오래 걸려도 앱이 무한 대기하지 않도록 watchdog 적용

## 검증

릴리스 전 검증:

- `pnpm typecheck`
- `pnpm test`
- `pnpm build`
- `pnpm dist:mac:free`
- ad-hoc codesign 검증
- ZIP 무결성 및 내부 구성 확인
- `/Applications/Markwand.app` 로컬 설치 후 실제 실행/종료 smoke
- bundle id Quit 3회 반복: `0.510s / 0.978s / 0.373s`

참고: `pnpm lint`는 현재 프로젝트에 ESLint 의존성/설정이 없어 실행 불가입니다. 릴리스 차단 이슈가 아니라 별도 품질 게이트 정비 항목으로 추적합니다.

## 알려진 제한

- 무료 ZIP 배포는 공증이 아니므로 첫 실행 시 macOS 경고가 표시됩니다. 정상 경로는 우클릭 또는 Control+클릭 → 열기입니다.
- Project Wiki는 현재 로컬 문서 메타데이터와 drift report 기반의 MVP입니다. 추후에는 더 깊은 의미 분석, 문서 간 요약, 사용자 정의 위키 페이지 생성을 확장할 예정입니다.
- GitHub Release는 prerelease로 배포합니다.
