# Markwand v0.3.0-beta.3 — DMG 창 원클릭 도움말

**배포일**: 2026-04-22
**배포 채널**: 내부 베타 (unsigned dmg + ad-hoc signing)
**이전 릴리스**: v0.3.0-beta.2 (2026-04-22)

기능 변경 없음. **DMG 창에서 바로 "시스템 설정" 으로 이동 가능** 하도록 설치 도움말 파일 내장.

---

## 🚀 설치 빠른 경로 (macOS 15+ / 26+)

1. 아래 [Assets](#assets) 에서 본인 Mac 기종의 DMG 다운로드 (Apple Silicon 은 `arm64`)
2. DMG 더블클릭 → 창이 뜹니다
3. `Markwand` 를 **Applications** 폴더로 드래그
4. Applications 에서 `Markwand` 더블클릭 → "악성 코드 확인 불가" 경고 → **완료** 클릭
5. **DMG 창의 `설치 도움말.webloc` 더블클릭** ← 시스템 설정 앱이 개인정보 보호 패널로 자동 이동
   (또는 브라우저에서 👉 **[시스템 설정 열기](x-apple.systempreferences:com.apple.settings.PrivacySecurity.extension)** 링크 클릭)
6. 스크롤 → `Markwand.app 이(가) 차단되었습니다` 옆 **그래도 열기** → Touch ID / 암호
7. Markwand 다시 더블클릭 → **열기** 확인

이후부터는 더블클릭만으로 바로 실행됩니다.

---

## 바뀐 것

- `build/설치 도움말.webloc` 신규 파일 — `x-apple.systempreferences:com.apple.settings.PrivacySecurity.extension` URL 링크
- `dmg.contents` 에 `.webloc` 위치 추가 (DMG 창 중앙 하단)
- DMG 창 크기 540×380 → 560×400 (.webloc 파일 포함 가시성)
- Spotlight 로 시스템 설정을 검색할 필요 없음 — DMG 창에서 클릭 한 번

## 변경 없음 (beta.2 와 동일)

- Ad-hoc 서명 (`afterPack` hook 으로 identifier `tech.spacewalk.markwand` 고정)
- `hardenedRuntime: false` (공증 없을 땐 해로움)
- 기능 전체 — SSH 원격 서버, 폴더 picker, 이미지 IPC, 한국어 UI 등
