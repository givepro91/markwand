# v0.4.0-beta.8 — corrected macOS free-install guidance

> 2026-05-01 · macOS prerelease (free ZIP, ad-hoc signed)

beta.8 supersedes beta.7.

## Why

beta.7 added `Open Markwand.command`, but local Finder testing showed the command file itself can be blocked by the same Gatekeeper quarantine flow. That made beta.7's fallback misleading.

## Changed

- Removed `Open Markwand.command` from the ZIP.
- Updated install docs and `First Run Guide.html` to say the verified free fallback clearly:

```bash
xattr -cr /Applications/Markwand.app
open /Applications/Markwand.app
```

## Verification

- `pnpm typecheck`
- `pnpm test`
- `pnpm dist:mac:free`
- ZIP content check: app + guide only, no command helper
- Quarantined local install smoke: Terminal fallback removes quarantine and launches Markwand

## Note

Without Apple Developer ID signing and notarization, a no-Terminal first launch cannot be guaranteed on recent macOS versions.
