# Markwand v0.3.0-beta.1 — 원격 SSH 서버 지원 베타

**배포일**: 2026-04-22
**배포 채널**: 내부 베타 (unsigned dmg)
**이전 릴리스**: v0.2.0 (2026-04-21)

Markwand 가 원격 SSH 서버의 마크다운 문서를 읽기 전용으로 탐색할 수 있게 되었습니다.
내 컴퓨터 밖에 있는 노트·문서 폴더도 앱 하나로 열람할 수 있습니다.

---

## ⚡ 주요 신규 기능

### 🌐 원격 SSH 서버 연결 (베타)

- **내 컴퓨터와 동일한 UX 로 원격 폴더 탐색** — SFTP 기반 읽기 전용, 서버에 아무것도 전송·변경되지 않습니다.
- **`~/.ssh/config` 호스트 자동 불러오기** — 이미 등록된 서버는 dropdown 에서 선택만 하면 host/port/user/key 자동 채움.
- **원격 폴더 picker** — POSIX 경로를 암기할 필요 없이 "폴더 탐색…" 버튼으로 서버 디렉토리를 직접 둘러보며 선택.
- **자동 로그인 (ssh-agent) 또는 키 파일 지정** — 터미널에서 쓰던 인증 방식 그대로.
- **서버 지문(TOFU) 확인 모달** — 최초 연결 시 SHA256 지문을 관리자에게 확인받고 신뢰할지 결정. 지문 변경 시 자동으로 연결 중단 (MITM 방어).
- **같은 서버 여러 폴더 등록 가능** — 프로젝트별로 워크스페이스를 나눠서 관리.

### 🖼 원격 이미지 뷰어

- 원격 마크다운 문서 내 상대경로 이미지(`./public/logo.svg` 등)가 실제로 렌더링됩니다 (IPC 스트리밍 경유).
- 이미지 파일을 좌측 트리에서 직접 클릭해도 뷰어에서 열립니다.
- 2MB 상한 + SVG/PNG/JPG/GIF/WEBP 6종 확장자.

### 🇰🇷 일반 사용자 친화 UI

- SSH 추가 모달 전면 한국어화 (host → 서버 주소, user → 계정 이름, ssh-agent → 자동 로그인 등).
- 에러 메시지 9종 자연어 매핑 (ECONNREFUSED · ETIMEDOUT · AUTH_FAILED 등 → "서버 주소가 맞는지 확인해주세요" 식).
- TOFU 모달 "신뢰하고 기억" + 영구 저장 고지.
- 워크스페이스 드롭다운: 🌐 `jay-dev / ccm_hub` 형태 (서버 별명 / 프로젝트 폴더).
- 로컬/SSH `<optgroup>` 시각 분리.

| 원격 프로젝트 스캔 | **BFS 병렬 readdir** — RTT × N → RTT × depth (실 원격 10~50배 체감 향상 추정) |

| 로컬 hot path | 바닥 기준선 대비 -33% ~ -49% 개선 (bench:transport 측정) |

## 🔒 보안

- **DC-4 bypass 0** — 저장된 서버 지문과 달라지면 `Trust` 버튼을 DOM 에서 **물리적으로 제거** (disabled 아님). bypass 통로 0.
- **읽기 전용 계약** — 원격으로 write 경로 일체 노출 안 됨.
- **ssh-agent + 키 파일 경로만 저장** — 키 내용 자체는 앱에 저장되지 않습니다.
- **루트 깊이 검증** — `/` 단독, `/home` 등 위험 경로 등록 차단.
- **ProxyJump 1-hop 백엔드 지원** — `~/.ssh/config` 의 ProxyJump 설정 인식 (UI 에서는 기본 필드만 자동 채움, 1.0 에서 UI 확장 예정).

## ⚠ 알려진 제한

- **베타에서 원격 워크스페이스는 "한 폴더 = 한 프로젝트"** 형태만 지원. "여러 프로젝트가 있는 상위 폴더" 모드는 RTT 비용이 커서 v1.0 에서 고급 옵션으로 재활성 예정.
- **원격 이미지 SVG/PNG 등은 2MB 상한**. 큰 이미지는 로드 실패 표시.
- **같은 서버에 2개 이상 워크스페이스 등록 시 연결 2개 생성** (v1.1+ 에서 연결 공유 최적화 예정).
- **원격 watcher 비배선** — 원격 파일 변경 실시간 반영 안 됨. 새로고침 또는 워크스페이스 재선택 필요.
- **drift 검증 · composer 토큰 추정 · 코드사이닝** — v1.0 스코프.

## 📂 외부 저장 항목

SSH 워크스페이스 등록 시 앱이 저장하는 값 (`electron-store` 로컬만):

- 호스트명 · 포트 · 계정명
- 키 파일 **경로** (내용 아님)
- 루트 경로 (폴더 선택값)
- TOFU 결정 시점의 SHA256 지문 + 알고리즘 + 최초 저장 시각

원격 서버로는 **어떤 데이터도 전송되지 않습니다**.
