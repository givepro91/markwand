# Markwand v0.3.0-beta.5 — 앱 아이콘 + 도움말 가독성

**배포일**: 2026-04-22

**이전 릴리스**: v0.3.0-beta.4 (2026-04-22)

기능 변경 없음. **앱 브랜드 아이콘 도입** + beta.4 의 설치 도움말 HTML 다크모드 가독성 개선.

---

## 🎨 새 앱 아이콘

기본 Electron 원자 아이콘 → **마크다운 M↓ 블루 그라디언트** (macOS 스타일 squircle).

- 배경: `#3B82F6 → #1D4ED8` 그라디언트 (Apple 공식 Blue 톤 범위)
- 마크: 흰색 `M` + 다운 화살표 (마크다운 공식 로고 영감)
- 모서리: 224px radius (macOS Big Sur+ squircle)

Dock, Launchpad, Finder 모두에서 즉시 마크다운 앱으로 인지됩니다.

## 🔧 도움말 페이지 다크모드 수정

beta.4 의 `여기를 먼저 더블클릭.html` 에서 macOS 다크모드일 때 **카드 배경은 흰색 + 텍스트는 밝은 회색** 으로 렌더되어 거의 안 보이던 문제.

- **원인**: CSS 규칙 순서 버그. `@media (prefers-color-scheme: dark) { .card { background: #2a2a2a } }` 가 일반 `.card { background: #fff }` **앞** 에 있어 cascade 에서 덮어써짐.
- **수정**: 모든 light 기본값 선언 → 맨 끝에 통합된 단일 dark media query 블록. 각 요소에 명시적 `color` 지정으로 body inherit 의존 제거.
