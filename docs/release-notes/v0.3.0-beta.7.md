## 🕐 최근 7일 문서 패널 (ProjectView)

좌측 사이드바, 프로젝트 헤더와 파일 트리 사이에 **최근 7일** 동안 수정된 문서를 한눈에 보여주는 별도 섹션.

- **위치**: 사이드바 헤더(프로젝트명 + Claude 버튼) 아래, 파일 트리 위
- **시각 구분**: 사이드바와 다른 배경 톤 + 위/아래 구분선(아래쪽 2px) + 시계 아이콘 + 카운트 배지
- **항목**: 파일명 + 우측에 "오늘 / 어제 / N일 전" 상대일자, 마우스를 올리면 절대일자 툴팁
- **활성 항목 강조**: 현재 보고 있는 문서는 좌측 액센트 막대 + 굵은 글씨
- **접기 / 펼치기**: 헤더 클릭 → 즉시 토글, 다음 실행에도 상태 유지
- **빈 상태**: 7일 내 수정된 .md 가 없으면 패널 자체가 사라짐 (시각 노이즈 0)
- **자동 새로고침**: 분 단위로 시각 라벨 갱신 (자정 경계에서 "오늘" → "어제" 자동 전환)
- 최대 10개까지, 초과분은 "+N개 더" 라벨로 안내 (전체 인박스는 ⌘K 또는 인박스 뷰 활용)

> 데이터 출처: 파일시스템 `mtime`. FilterBar 의 날짜 필터(오늘/7일/30일)와 독립 — 패널은 항상 raw `mtime` 기준 7일을 보여줍니다.

## 🏷 이름 통일 (md-viewer → Markwand)

beta.6 까지 일부 영역에 남아 있던 개발 시점 슬러그 `md-viewer` 를 사용자 노출 + 내부 식별자 모두 **Markwand** 로 정리.

- **윈도우 타이틀**: `md-viewer` → `Markwand`
- **README 제목**: `md-viewer` → `Markwand`
- **electron-store 데이터 파일명**: `md-viewer.json` → `markwand.json`
  - 폴더 경로(`~/Library/Application Support/Markwand/`)는 그대로
  - **⚠️ 기존 prefs(워크스페이스 등록·읽음 이력·뷰 모드 등)는 1회 초기화됩니다**. 첫 실행 후 워크스페이스를 다시 추가해주세요.
- CSS Highlight API 식별자(`md-viewer-find-match` → `markwand-find-match`) — 내부 변경, 영향 없음

> 보존: 과거 스프린트 문서 슬러그(`docs/plans/md-viewer-mvp.md` 등)는 히스토리 보존 차원에서 그대로 둡니다.

## 🌐 i18n 잔여 케이스

beta.6 의 i18n 작업에서 누락된 6건 추가 + 메시지 정합화.

- 토스트 3건: "선택에서 제거", "마지막 선택 복원", "⌘K 검색 안내"
- ProjectView 빈 상태 설명 ("트리에서 .md 또는 이미지 파일을 클릭하면 …")
- 전체 프로젝트 그룹 섹션 aria-label
- SSH 비활성 안내 메시지 안의 영문 라벨이 실제 설정 UI("베타 기능 → 원격 SSH 서버 연결")와 안 맞던 것 정합화

이제 ko/en 양쪽 모두 사용자 노출 라벨이 100% 번역 자원에서 공급됩니다 (DevTools 콘솔 메시지 5건 제외 — 비노출).
