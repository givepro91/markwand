## 🛠 Mermaid 다이어그램 박스 내부 글자 짤림 해소

문서에 mermaid 플로우차트/시퀀스 다이어그램이 있을 때, 노드 박스 내부 텍스트 끝 글자가 박스 밖으로 밀려 잘리던 현상 수정.

- **원인**: `mermaid.initialize` 에 `startOnLoad` 와 `theme` 만 지정하고 `flowchart` / `themeVariables` 옵션을 주지 않아, 노드 자동 크기 계산이 실제 렌더 글꼴과 불일치
- **수정**: 루트 `htmlLabels: true` + 앱 CSS와 동일한 `fontFamily`/`fontSize` + `flowchart.{padding, nodeSpacing, rankSpacing, diagramPadding, useMaxWidth}` 일관 주입. 다크 테마 전환 시에도 동일 옵션 유지
- **CSS 가드**: `.mermaid-block svg { max-width: 100%; height: auto; }` + `foreignObject { overflow: visible }` 로 긴 다이어그램은 컨테이너 가로 스크롤, 짧은 다이어그램은 가운데 정렬

## 🔄 프로젝트·문서 변경 자동 반영 복구

**v0.1 이후 로컬 파일 watcher 가 비활성 상태**였던 점을 복구. 이제 로컬 워크스페이스에서 일어나는 변경이 자동으로 UI 에 반영됩니다.

- **파일 변경**: `.md` 또는 이미지 파일 추가/수정/삭제 시 열려 있는 프로젝트의 파일 트리·문서 리스트 즉시 반영 (150ms debounce)
- **프로젝트 변경**: 워크스페이스 루트 하위 depth ≤ 2 디렉토리가 새로 생기거나 제거될 때 프로젝트 목록 자동 갱신 (500ms debounce + 렌더러 2s 쓰로틀로 watcher storm 수렴)
- **방어 장치**: `node_modules`, `.git`, `dist`, `build`, `.next` 등 무거운 디렉토리는 이전과 동일하게 watch 제외 → IPC 폭발 회피. `.md`/이미지 외 파일은 필터링
- **수동 새로고침**은 그대로 남아있음 (fallback)
- **원격(SSH)** 워크스페이스는 기존 SshPoller 30s 주기 유지 (별도 후속에서 디렉토리 갱신 추가 예정)

## 🖼 최근 7일 패널에 이미지 탭 추가

프로젝트 뷰 좌측 "최근 7일" 패널에 **문서 / 이미지** 두 탭이 추가되었습니다.

- 섞이지 않고 탭으로 구분 — 최근 편집한 md 문서와 이미지를 각각 10개씩 별도 관리
- 마지막 선택한 탭은 앱 재시작 후에도 유지 (prefs `recentDocsTab`)
- 양쪽 탭 모두 비어 있으면 패널 자체가 숨겨져 시각 노이즈 0 (기존 동작 유지)
- 한쪽 탭만 비면 "최근 수정한 이미지가 없습니다" 같은 안내 문구
- 접근성: WAI-ARIA Tabs 패턴 준수 (`role="tablist"` + `role="tab"` + `role="tabpanel"` + `aria-selected` + `aria-controls` + 화살표 키 네비게이션)
