## üïê Recent 7 days panel (ProjectView)

A dedicated section between the project header and the file tree that shows every document touched in the last 7 days at a glance.

- **Location**: sidebar, between the project header (name + Claude button) and the file tree
- **Visual separation**: different background tone from the sidebar + 2px bottom divider + clock icon + count badge
- **Item**: filename + relative date on the right ("Today / Yesterday / N days ago"), with an absolute-date tooltip on hover
- **Active highlight**: the currently-open doc gets a left accent bar + bold text
- **Collapse / expand**: click the header to toggle ‚Äî state persists across sessions
- **Empty state**: if nothing has been modified in the last 7 days, the entire panel hides (zero visual noise)
- **Auto-refresh**: relative labels re-render every minute ("Today" ‚Üí "Yesterday" flips automatically at midnight)
- Up to 10 items ‚Äî overflow collapses to a "+N more" label (use ‚åòK or the inbox view for the full list)

> Data source: filesystem `mtime`. This panel is independent from the FilterBar date filter (Today / 7d / 30d) ‚Äî it always shows raw `mtime` within 7 days.
